### Памятка


#### 1. Клонируйте репозиторий на свой компьютер

Деплой: npm -i
```
npm -i
```
Сайт бедет в папке сборки:
```
build
```
Команда клонирует репозиторий на ваш компьютер и подготовит всё необходимое для старта работы.
